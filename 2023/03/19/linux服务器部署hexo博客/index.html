服务器搭建Nginx

  1. 以CentOS为例

    1. 安装

    2. 查看是否安装nginx

    3. 开启nginx

      1. systemctl start nginx开启
      2. systemctl stop nginx关闭

    4. 将本地blog博客下文件移到服务器

    5. 接下来配置nginx链接到网站源文件，主要通过修改nginx目录下的nginx.conf文件实现。（我的nginx安装目录是/usr/local/nginx如果你找不到可以用whereis nginx命令看一下。）配置文件内容繁多，不一一说明，有需要的时候搜索下即可。

      1. cd到nginx目录。
      2. vim nginx.conf进入编辑文件（不会用vim的花几分钟了解下），找到http大括号下的sever大括号部分，修改location /里面的root xxxx为root 你的网站目录

    6. nginx -s reload重启
