博客搭建

使用node.js框架与github平台搭建

接下来就是我的搭建流程，还是比较麻烦，谢谢一些b站朋友分享自己的搭建过程。

Node.js下载与安装

Node.js的官网：https://nodejs.org 点击下面的下载就是行了，一直点next就行，中间的安装地址记得自己改一下，不然默认放在C盘，不过问题不大。

安装后，Win+R打开cmd窗口，输入node -v,如果有版本信息就是安装好了，npm -v 同理，这样就好了一部分

如果是该命令不是语法，就是环境变量没配好，一般都是自动配好的。首先找设置->系统->关于->高级系统设置（右边）->高级->环境变量 （以win10为例） 或者直接左下角搜索高级系统设置

找到后，点编辑，增加地址，地址就是你之前安装路径

接下来就是利用npm来安装Hexo博客框架，直接在cmd窗口中输入

npm install -g cnpm –registry=https://registry.npm.taobao.org

可能安装有点慢，如果一直安装不下来，就退出窗口重新来一遍

安装后，就可输入cnpm -v看版本号，如果有就是安装完成（不完成，我也不会）
